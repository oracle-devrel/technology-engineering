# OIC and OCI Queue Service Interactions

Asset that contains oic project with all the basic interactions to use OCI Queue Service.


## OIC OCI_QUEUE_DEMO-OCI_QUEUE_DEMO_PROJECT

This OIC project provides solutions for automating message processing and queue management, benefiting industries such as cloud, financial, and technology, by enabling real-time data processing, efficient queue management, and event-driven business object processing.

Review Date: 19.01.2026

# When to use these assets?

These assets should be used whenever needed to orchestrate publishing and consuming messages from/to OCI queues, triggering transformations, mappings, and events, leveraging REST APIs and scheduled orchestration to facilitate event-driven business object processing and queue management.
