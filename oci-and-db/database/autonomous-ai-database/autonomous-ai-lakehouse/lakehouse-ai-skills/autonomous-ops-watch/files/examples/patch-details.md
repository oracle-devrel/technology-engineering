# Example: Patch Details

## User prompt

What was fixed in patch version ADBS-25.4.4.2?

## Expected response style

Source: `DBA_CLOUD_PATCH_INFO`

Summarize visible bug fixes for the requested patch version, grouped by component when helpful. Mention that this view may not be complete release notes for every internal patch change.

If no rows are visible, say that no patch detail rows were visible for the connected user and patch version, not that the patch had no changes.
