# Integrations

Focuses on integrating an OpenShift cluster running on OCI with OCI services and common platform add-ons. It describes practical patterns for storage, logging, and migration use cases, such as using OCI Object Storage, deploying Loki for log aggregation, and enabling tools like MTV or MTC. The section emphasizes configuration steps, required permissions, and operational considerations so integrations remain secure, reliable, and maintainable over time.
