# Assets

Contains shared visual and supporting materials used by the OpenShift on OCI documentation. It typically includes architecture diagrams, network topology sketches, flow charts for installation and Day 2 procedures, and small example snippets or templates referenced by multiple guides. Keeping these assets centralized ensures the documentation stays consistent, avoids duplication, and makes it easy to update diagrams or example files without hunting through many folders.
