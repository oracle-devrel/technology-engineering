# Documents

This directory is the central documentation hub for everything related to running OpenShift on Oracle Cloud Infrastructure. It is organized by lifecycle phase so readers can move from overview and scope, to architecture patterns and reference designs, to installation methods and prerequisites, and finally into integrations and Day 2 operations. The intent is consistency: each topic should describe purpose, assumptions, required inputs, and validation steps, so procedures can be repeated across environments without relying on tribal knowledge. Integrations capture practical OCI service usage and platform add-ons, while Day 2 focuses on operating tasks such as upgrades, scaling, node onboarding, and standards enforcement. Shared diagrams and examples referenced by the documentation are stored alongside to keep the docs complete and easy to navigate.
