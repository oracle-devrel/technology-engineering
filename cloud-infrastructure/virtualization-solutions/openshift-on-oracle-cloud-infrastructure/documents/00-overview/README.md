# Overview

The starting point for OpenShift on OCI and explains how this repository should be used across the cluster lifecycle. It introduces the scope, key assumptions, and terminology that matter in OpenShift deployments, and it links to authoritative Red Hat and Oracle sources. The section helps readers choose the right path into architecture and installation topics and understand which guidance is OpenShift platform specific versus OCI integration specific.
