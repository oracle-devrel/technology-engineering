# Installations

Describes the supported installation paths for deploying OpenShift on OCI and the prerequisites required for each method. It covers the inputs and validations needed to build a reliable cluster, compares approaches such as Terraform-based provisioning, Assisted Installer, agent-based workflows, and fully controlled manual installations
