# Scripts / OCI

The directory contains OCI-focused scripts and code snippets that standardize recurring platform tasks required to run OpenShift on Oracle Cloud Infrastructure. Typical content covers IAM setup (compartments, dynamic groups, policies, tagging), networking foundations (VCNs, subnets, route tables, gateways, NSGs/security lists, DRG patterns), and validation commands to confirm the expected state. The intent is repeatability and clarity: small, parameter-driven helpers that reduce manual errors, document “known good” procedures, and provide consistent checks across development, test, and production environments.
