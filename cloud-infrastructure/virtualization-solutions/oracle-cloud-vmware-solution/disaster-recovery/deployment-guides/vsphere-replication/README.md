# Oracle Cloud VMware Solution - vSphere Replication 9.x

This repository contains a detailed guide for using vSphere Replication 9.x with Oracle Cloud VMware Solution.

Reviewed: 21.10.2025

# When to use this asset?

This document serves as an asset for individuals and organisations seeking to deploy a disaster recoevery of their VMware estate in Oracle Cloud VMware Solution, to protect their critical applications to the newly deployed cloud environment using vSphere Replication.

# How to use this Asset

Use the README file in the files folder as a starting point for deploying a DR using vSphere Replication with OCVS. It includes architecture diagrams, and configuration steps for the solution.
