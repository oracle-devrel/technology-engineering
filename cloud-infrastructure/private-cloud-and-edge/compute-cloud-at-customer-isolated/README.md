# Compute Cloud@Customer Isolated

Reviewed: 16.10.2025

Oracle Compute Cloud@Customer Isolated is a fully air-gapped, on-premises cloud solution that delivers the same comprehensive compute, GPU, storage, and networking capabilities as Oracle Compute Cloud@Customer in a highly secure and isolated environment.
