# Compute Cloud@Customer

Updated: 23.12.2025

Oracle Compute Cloud@Customer is fully managed, rack-scale infrastructure that allows organizations to use Oracle Cloud Infrastructure (OCI) Compute anywhere. It provides the benefits of cloud automation and economics in customer data centers by running OCI Compute and GPU shapes with storage and networking services on Compute Cloud@Customer.
