# Oracle Goldengate

Oracle GoldenGate provides real-time capture, transformation, routing, and delivery of database transactions across heterogeneous systems and cloud. The software facilitates high performance, lowimpact data movement with low latency to a wide variety of databases, platforms, and clouds while maintaining transaction integrity. Using Oracle GoldenGate with Oracle Streaming Analytics allows users to process and analyze large scale real- time information by using sophisticated correlation patterns, enrichment, and machine learning algorithms.
