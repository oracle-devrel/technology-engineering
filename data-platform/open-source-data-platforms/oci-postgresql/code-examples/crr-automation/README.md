# Automate Switchover/Failover operations with OCI PostgreSQL Warm-Standby DR

Reviewed: 7.04.2026

# When to use this asset?

When you are looking to orchestrate the warm-standby functionality via APIs for OCI Database with PostgreSQL

# How to use this asset?

Open the crr-warm-standby-api.md file and follow the steps carefully.

# Pre-requisites:

- Oracle Linux VM
- OCI PostgreSQL instance, with warm-standby configured*.
