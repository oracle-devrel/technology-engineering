# Open Source Data Platforms 

Open Source Data Platforms continue to extend the Oracle Data Platform with Key Open Source Offerings including:

- Big Data Service
- OCI PostgreSQL
- OCI Cache Redis
- OCI Data Flow (Apache SPARK)
- OCI Streaming (Kafka)
- OCI OpenSearch
