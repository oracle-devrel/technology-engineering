# Generative Development with APEX

Oracle APEX generative AI capabilities enhance the low-code application development experience. These features aim to accelerate development, simplify complex tasks, and enable richer, more intelligent applications.


This area focuses on generative development patterns with Oracle APEX, including : 
- APEXlang
- AI-assisted application generation
- AI Assistant in Code Editors
- AI Interactive Reports
- AI Agents on APEX
- Vector Search Support
- integration patterns with Oracle AI services.

# APEX 26.1 Generative Development Features

Oracle APEX 26.1 introduces new AI-powered development capabilities for building trusted enterprise applications. Key areas include:

- APEXlang for declarative, reviewable, source-controlled application specifications
- AI-assisted application generation
- AI Interactive Reports for natural language report exploration
- AI Agents and AI Tools for governed conversational actions inside APEX applications
- Generative AI support in application development workflows
- Support for structured development with SQL Developer for VS Code and SQLcl
