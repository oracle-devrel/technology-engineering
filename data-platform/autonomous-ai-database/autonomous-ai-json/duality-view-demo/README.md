This repository contains demo of using JSON-Relational Duality Views

- duality_views.sql is a SQL*Plus script demonstrating using JSON-Relational Duality Views in SQL and SODA
  This script should be used as the first one, as it demonstrates how to create views
- duality_views.js is a mongosh.script demonstrating using JSON-Duality Views in Oracle API for MongoDB
