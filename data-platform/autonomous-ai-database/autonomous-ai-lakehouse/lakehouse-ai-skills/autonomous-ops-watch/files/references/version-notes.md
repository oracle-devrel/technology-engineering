# Version Notes

## v0.1

Initial version of Autonomous AI Lakehouse Ops Watch.

Scope:

- One connected Oracle Autonomous AI Lakehouse.
- SQL/MCP-first execution.
- Read-only evidence only.
- No OCI API, OCI Events, Console, CLI, SDK, Resource Principal views, or lifecycle actions.
- No external scheduler or ETL job inspection.
- No mutating SQL.

Included capabilities:

- Maintenance reader.
- Patch details inspector on request.
- Lockdown profile error analyzer.
- Client error analyzer.
- Operator access auditor.
- Unified audit reviewer.
- Session activity and blocking snapshot.
- Invalid objects and compile errors.
- Table and partition access stats on request.
