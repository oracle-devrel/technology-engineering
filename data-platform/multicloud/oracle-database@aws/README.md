# Oracle Database@AWS

Oracle Database@AWS is a collaborative service between Oracle and Amazon, delivering Oracle AI Database capabilities, fully managed by Oracle within AWS data centers. This solution allows organizations to migrate Oracle database workloads, including those using Oracle Real Application Clusters, to AWS and enables seamless data integration across Oracle and AWS environments with zero-ETL processes. 
Available offerings include Oracle Exadata Database Service on Dedicated Infrastructure and Oracle Autonomous AI Database on Dedicated Exadata Infrastructure. 
Currently, Oracle Database@AWS is available in two Availability Zones in both the US East (N. Virginia) and US West (Oregon) regions.
