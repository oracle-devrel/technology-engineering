# Replacing transcievers on an activated system

For systems using fibre PCIe cards this can be requested. If the customer has 10 Gbps or 25 Gbps capable only transceivers and the data center network has been changed Oracle can replace those transcievers following a service request submitted by the customer. Please note that the procedure is not automatic and requires an approval from Oracle side. 

# ExaDB-C@C Long Range Transceivers for use with Single Mode Optics

Long range transceivers (LR SFP+/SFP28) for ExaDB-C@C is not part of the standard delivery, and as such not available only following a special request.

For customers standardizing on single mode optics inside their datacenters, we can evaluate case by case the delivery of LR SFP+/SFP28.
The recommendation is to not use these for long distances (metro / kilometers) between DB servers and customer switches.
