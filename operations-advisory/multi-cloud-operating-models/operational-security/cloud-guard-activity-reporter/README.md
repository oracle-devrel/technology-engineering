# Cloud Guard Activity Reporter

Last reviewed: 2026-06-04

---

## Overview

Cloud Guard Activity Reporter is a Go tool to track, summarize, and export Oracle Cloud Guard activity in seconds.

Features:
* Filter by compartment, region, resource type, or risk level 
* Generate CSV reports 
* Quick insights on recent security problems

Check it out [here.](https://github.com/eugsim1/cloudguard-activity-reporter)
