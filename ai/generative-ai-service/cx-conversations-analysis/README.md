
# Customer Experience Conversations Analysis

The following example transcribes audio from Call Centers using OCI Speech and then makes an analysis using OCI Generative AI Service with OpenAI gpt-oss models.

Example to test it out: [Gridspace-Stanford Harper Valley Dataset](https://github.com/cricketclub/gridspace-stanford-harper-valley)
 
Reviewed: 11.12.2025

Authors: Yainuvis Socarras and Cristina Granes
